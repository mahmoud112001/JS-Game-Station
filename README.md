## 🃏 Three Games

### ⚙️ v1 — The Scaffold *(Hangman Classic)*
> *"Where it all began. Raw, minimal, and pure."*
- Light pastel theme
- 4 categories
- Fixed 8 lives
- Single HTML file

### ⭐ v2 — Gallows Noir *(Hangman Dark Edition)*
> *"Reborn in darkness. Harder, smarter, and alive with animation."*
- Dark animated theme
- 6 categories + difficulty system
- Leaderboard via `localStorage`
- Physical keyboard support
- Separated HTML + CSS + JS

### ♟️ Grid Wars *(Tic Tac Toe)*
> *"Three in a row wins it all. Challenge a friend or take on an unbeatable AI."*
- Player vs Player + Player vs CPU
- 3 difficulty levels — Easy · Medium · Hard
- Hard mode uses full **Minimax AI** (unbeatable)
- Score persistence via `localStorage`
- Built with TypeScript + SCSS + Vite

---

## ✨ Landing Page Features

- Animated background grid + floating glowing blobs
- 22 floating particles in 4 colors (CSS + JS)
- Shimmer effect on the JS GAMES title
- Three interactive game cards with hover animations
- Corner bracket decorations per card
- Each card has a unique color identity (blue · yellow · green)
- Mini 3×3 board decoration on the Tic Tac Toe card
- Credentials section — Course · Instructor · Program · Author
- GitHub + LinkedIn social buttons
- Fully responsive — 3 cols → 2 → 1
